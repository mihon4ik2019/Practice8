1. **Установите Node.js** (если не установлен): 
   https://nodejs.org/
2. **Клонируйте репозиторий**
   ```sh
   git clone <URL_REPOSITORY>
   cd <REPOSITORY_FOLDER>
   ```
3. **Установите http-server** (если не установлен):
   ```sh
   npm install -g http-server
   ```
4. **Запустите сервер**
   ```sh
   http-server -p 8080
   ```
5. **Откройте в браузере**
   http://localhost:8080
